VBRS TV Kodi Repository

Willkommen beim offiziellen Kodi-Addon Repository von VBRS TV 🎬

Dieses Repository enthält eine Sammlung eigener Tools, Utilities und Module zur Optimierung und Erweiterung von Kodi – insbesondere für Android- und FireTV-Geräte.

---

📦 Enthaltene Addons

🔐 VPN Starter

- ID: "plugin.program.vpnstarter"
- Lädt automatisch ".ovpn"-Konfigurationsdateien
- Speichert Dateien unter:
  "/storage/emulated/0/Download/Vpnstarter/"
- Öffnet anschließend eine externe App (z. B. X-plore)
- Plattform: Android / FireTV

---

🔐 VPN Starter DE

- ID: "plugin.program.vpnstarterde"
- Deutsche Variante des VPN Starters
- Unterstützt externe App-Ausführung
- Optimiert für Android

---

⚡ VBRSTuning Tool

- ID: "script.vbrstuning"
- Optimiert Kodi Buffer / RAM Cache
- Speziell für FireTV / FireStick entwickelt
- Ziel: Reduzierung von Buffering-Problemen

---

🧰 VBRS Tools Module

- ID: "script.module.vbrs.tools"
- Zentrale Library für andere VBRS Addons
- Features:
  - DNS-over-HTTPS (Cloudflare, Google, Quad9)
  - VPN-Statusprüfung
- Mehrsprachig: DE / EN / IT

---

🧹 Simple Cache Cleaner

- ID: "plugin.program.cachecleaner"
- Löscht Kodi Cache schnell und einfach
- Zeigt Speicherplatz an
- Mehrsprachig (DE / EN / IT)
- Optimiert für Android / FireStick

---

📤 Addon Exporter

- ID: "plugin.program.addonexporter"
- Exportiert installierte Kodi Addons als ZIP
- Inklusive Dependencies
- Ideal für Backups oder Weitergabe

---

⚙️ Installation

Manuelle Installation

1. Öffne das Repository:
   https://github.com/Vbrscrushtv/vbrstv

2. Lade die gewünschte ZIP-Datei herunter

3. In Kodi:
   
   - Gehe zu Addons
   - Wähle "Aus ZIP-Datei installieren"
   - Wähle die Datei aus
